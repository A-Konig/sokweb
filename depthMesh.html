Depth Mesh

O aplikaci

Aplikace Depth Mesh umožňuje uživateli seznámit se s pokročilými možnostmi snímání hloubky kamerou RealSense. Je zde možné nastavit jednotlivé parametry snímání a ze snímaného hloubkového obrazu je vytvořen trojrozměrný model, na který je jako textura aplikován barevný snímek scény. Dále aplikace umožňuje úpravu tohoto modelu pomocí uživatelem zadaného kódu. Vzniklý model lze zaslat na server pro další zpracování.

Aplikace spolupracuje se serverem, na který je možné zaslat vytvořený 3D model scény, nebo zapnout automatické snímání a odesílání. Při připojení k serveru musí zadané url končit lomítkem a jméno klienta musí být unikátní pro všechny připojené klienty typu Depth Mesh.

Instalace

Z odkazu níže (tlačítko „Stáhnout”) lze stáhnout nejnovější release aplikace Depth Mesh. Aplikace se spouští dvojklikem na soubor DepthMesh.bat, který se nachází v archivu DepthMesh.zip. Návod na zacházení s aplikací a příklady úloh pro studenty jsou dostupné níže.

Zdrojové kódy aplikace jsou také dostupné z odkazu níže (tlačítko „Stáhnout”). Nacházejí se v archivu Source code. Jedná se o zdrojové kódy VisualStudio projektu v jazyce C#. Pro úspěšný setup projektu je třeba splnit následující předpoklady:

  • nainstalovaný .NET 6,
  • nainstalovaný cmake,
  • nainstalovaná knihovna Intel RealSense SDK 2.0.
Dále lze setup provést podle následujícího postupu
  1. Cesta k RS knihovně se nastaví v souboru Depth_Mesh/ZCU.TechnologyLab.DepthClientLib/CMakeLists.txt (nyní je tam nastavena defaultní - C:/Program Files (x86)/Intel RealSense SDK 2.0)
  2. Dále se setup projektu provede spuštěním bat skriptu setupDepthMesh.bat, nebo manuálně podle následujícího postupu:
    Ve složce Depth_Mesh provést následující příkazy
    • curl -O https://librealsense.intel.com/rs-tests/TestData/stairs.bag -k
    • cd ./ZCU.TechnologyLab.DepthClientLib
    • cmake .
    Do složky "vendor" na stejné úrovni jako DepthClient.sln nakopírovat do odpovídajících složek zdrojové kódy knihovny Common a serveru
    • do common nakopírovat složku ./Common/ZCU.TechnologyLab.Common
    • do server složku ./Server/ZCU.TechnologyLab.VirtualWorldServer
  3. Spustit sln projekt ve Visual Studiu 2022
  4. Nastavit jako projekt po spuštění ZCU.TechnologyLab.DepthClient

Zrojové kódy lze upravovat nebo použít pro vývoj vlastních aplikací, ale není dovoleno odstraňovat podpis původního autora.

Konfigurační soubor

Konfigurační soubor s názvem config.txt musí být umístěn ve složce src, ve které se mimo jiné nachází exe soubor aplikace. Z konfiguračního souboru se načítají počáteční hodnoty parametrů aplikace. Zadané url serveru musí končit lomítkem a jméno musí být unikátní pro všechny klienty typu Depth Mesh. Jeho tvar je následující: 

    < cesta k python .dll >
    < url serveru >
    < jméno klienta >
